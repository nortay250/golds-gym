# Fitness App

- React JS, TypeScript
- Styling: Material UI v5
- Rest API: Rapid API Exercise DB and Rapid API Youtube Search

## Functionality

- Fully responsive
- Select workouts from different exercise categories and specific muscle groups
- Play workout videos from Youtube
